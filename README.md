# Ads CPC Forecast

A simple web tool to estimate advertising clicks, CPC, CTR and campaign performance scenarios.

## Features

- Estimate average CPC
- Calculate CTR from impressions and clicks
- Simulate positive, neutral and negative CPC scenarios
- Display forecast charts
- Runs directly in the browser
- No account required
- No server-side storage

## Privacy

This tool runs locally in the user's browser.

The values entered by the user are not intentionally sent to external services, stored on a server, or logged by this project.

## Disclaimer

This project is provided for educational and informational purposes only.

The results are simplified estimates and do not guarantee real advertising performance. Actual CPC, CTR, conversions, costs and campaign results may vary depending on platform, targeting, competition, quality score, market conditions and many other factors.
